## 고칠 점 & 한마디
이 문제 특이한 점은 총 두 가지 였다.
1) 공통된 string의 길이가 K보다 커도 상관없다고 했는데, 사실 K보다 큰 공통 문자열이 있다면 K길이만 조사해도 될 것이다. 그러므로 K 길이를 가지는 문자열만 탐색하면 된다.
2) 모든 경우의 수에 대해서 탐색을 해도, 제한시간 안에 돌아간다는 게 포인트 (O(NM))
